# Hash-Brute-Force
This is a js code to return the Hash with the digits you like at the end


1. RUN `npm install`.

2. Set your own string (line 26 & 35).

3. Set your own digits (line 42).

4. RUN `node app`.

5. Also you can change your Hash type (line 38)
